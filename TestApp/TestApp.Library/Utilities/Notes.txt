Install EF Core:
dotnet tool install --global dotnet-ef


Move to folder Library with cd command (cd .\TestApp.Library):
dotnet add package Microsoft.EntityFrameworkCore.Design
dotnet add package Microsoft.EntityFrameworkCore.SqlServer

Migrate database to code:
dotnet ef dbcontext scaffold "Data Source=sql5097.site4now.net;Initial Catalog=DB_A49E03_testapp;user id=DB_A49E03_testapp_admin;password=testapp123456789" Microsoft.EntityFrameworkCore.SqlServer -f --context TestAppEntities --use-database-names -o DAL/Models
